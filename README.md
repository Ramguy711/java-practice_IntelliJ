# Java Practice 🚀

A collection of Java programs written while learning Core Java fundamentals.

## Topics Covered
- Variables, Data Types, Operators
- Control Flow (if/else, switch)
- Loops (for, while, do-while, nested)
- Methods and Varargs
- Arrays and Strings
- OOP (Classes, Objects, Inheritance, Abstract Classes)
- Mini Projects

## Mini Projects
| Project | Description |
|--------|-------------|
| Banking Program | Deposit, withdraw, check balance |
| Slot Machine | Betting game with emoji symbols |
| Stone Paper Scissors | Player vs Computer with score tracking |
| Quiz App | 5-question multiple choice quiz |
| Guess The Number | Random number guessing with attempts |
| Dice Roller | ASCII art dice display |

## Tools Used
- Java 17+
- IntelliJ IDEA
